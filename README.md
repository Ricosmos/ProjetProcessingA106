# Projet-Processing-A106

Ce dépôt contient le code source du projet de processing dans le cadre de la première année du master IWOCS à l'Université du Havre. 

Le projet vise à modéliser la salle de classe A106 en utilisant le langage de programmation Processing.

## Notes

- Les mesures sont en centimètres.
- L'origine du repère est située en bas à gauche de la salle.
- Les constantes de tailles sont divisées par 2 pour définir le rayon des objets, la mesure complète étant le diamètre.
